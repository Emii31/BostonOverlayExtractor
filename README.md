# Boston Overlay Extractor (BOE)

Boston Overlay Extractor is a professional Android firmware analysis toolkit
designed to automatically generate Runtime Resource Overlays (RRO) and
Magisk compatibility modules for Android GSIs.

Current Status:

- Bootstrap
- Under Development

Target Platform

- Motorola Boston (parrot)
- Android 15 Vendor
- Android 16 GSI

Future Support

- Samsung
- OnePlus
- Xiaomi
- Pixel
